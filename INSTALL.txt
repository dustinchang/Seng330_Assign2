#Tools needed to build project
  Git
  Google protobuf library

#Instructions to build project
  -Download or clone repository files from GitHub https://github.com/dustinchang/Seng330_Assign2
  -Navigate to the src directory
  -Run 'make'
  -Then you can run ./main_cpp which is where all the execution of prototype class design takes place and the addition of protocol buffer data is
  -Then run ./list_portfolio_cpp to list all the stored data in the protocol buffers
